OWENS
Owen's T Function

OWENS is a C library which evaluates Owen's T function.

Licensing:

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Languages:

OWENS is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version

Related Data and Programs:

ASA005, a C library which evaluates the CDF of the noncentral T distribution, and uses Owen's T function.

ASA076, a C library which evaluates Owen's T function.

ASA243, a C library which evaluates the CDF of the noncentral T distribution.

TEST_VALUES, a C library which includes selected values of many special functions.

TOMS462, a FORTRAN77 library which evaluates the bivariate normal distribution;
this is ACM TOMS algorithm 462.

Reference:
