PARTITION_PROBLEM
The Partition Problem

PARTITION_PROBLEM is a C library which seeks solutions of the partition problem, splitting a set of integers into two subsets with equal sum.

Languages:

PARTITION_PROBLEM is available in a C version and a C++ version and a FORTRAN90 version and a FORTRAN77 version and a MATLAB version.
