PLATO_PLY
Write a Platonic Solid to a PLY File

PLATO_PLY is a C program which writes any one of the five Platonic solids to a PLY file, by Greg Turk.

Usage:

plato_ply -switch > file.ply
where
  • switch chooses the solid:
    • c: the cube
    • d: the dodecahedron
    • i: the icosahedron
    • o: the octahedron
    • t: the tetrahedron
  • file.ply is the name of the output PLY file.
