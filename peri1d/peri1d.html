PERI1D
Peridynamics for 1D Time-Dependent Problem

PERI1D is a C program which sets up and solves a 1D time-dependent peridynamics problem, by Miroslav Stoyanov;

Licensing:

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Languages:

PERI1D is available in a C version.

Related Data and Programs:

PERI2D, a C program which sets up and solves a 2D time-dependent peridynamics problem, by Miroslav Stoyanov;

PERIDYNAMICS_1D_STEADY, a MATLAB library which solves a 1D steady version of the Poisson equation, using the non-local peridynamics model, by Marta D'Elia.
