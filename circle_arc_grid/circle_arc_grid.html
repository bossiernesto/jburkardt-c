CIRCLE_ARC_GRID
Compute Grid Points Along a Circular Arc

CIRCLE_ARC_GRID is a C library which calculates a grid of points equally spaced along a circular arc.

Licensing:

Languages:

CIRCLE_ARC_GRID is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version.

Related Programs:

BALL_GRID, a C library which computes grid points that lie inside a 3D ball.

CIRCLE_GRID, a C library which computes grid points that lie inside a circle.

ELLIPSE_GRID, a C library which computes grid points that lie inside an ellipse.

ELLIPSOID_GRID, a C library which computes grid points that lie inside a 3D ellipsoid.

TETRAHEDRON_GRID, a C library which computes a tetrahedral grid of points.

TRIANGLE_GRID, a C library which computes a triangular grid of points.

ARC is a set of 10 equally spaced points from 0 to 90 degrees, on a circle of radius 2 centered at (5,5).

  • arc.txt, the data.
  • arc.png, an image of the points.

PENTAGON is a set of 6(!) equally spaced points from 0 to 360 degrees. This is the easiest way to get a pentagon, but you should really edit the file and remove the last point, since it repeats the first one. The alternative is to specify that the angles range from 0 to 288 degrees!

  • pentagon.txt, the data
  • pentagon.png, an image of the points.
