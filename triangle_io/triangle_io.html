TRIANGLE_IO
Read and Write TRIANGLE Files

TRIANGLE_IO is a C library which can read or write some simple versions of some of the files used by Jonathan Shewchuk's triangle program.

Languages:

TRIANGLE_IO is available in a C version and a C++ version.

Related Data and Programs:

POLY, a data directory which contains a description and examples of the POLY file format.

SHOWME, a C program which can display the POLY files uses as input to TRIANGLE, and the output files that define meshes and other objects.

TRIANGLE, a C program which computes Voronoi diagrams and Delaunay triangulations, and creates and manipulates files that can be displayed by SHOWME.

TRIANGLE_DISPLAY, a MATLAB program which displays the nodes and elements of a triangulation on the MATLAB graphics screen, assuming the data has been stored in NODE and ELE files by the TRIANGLE program.

TRIANGLE_FILES, a data directory of examples of files used by the triangle and showme programs.

List of Routines:

  • ELEMENT_DATA_EXAMPLE returns the element information for the example.
  • ELEMENT_DATA_READ reads the data from an element file.
  • ELEMENT_SIZE_EXAMPLE returns the element size information for the example.
  • ELEMENT_SIZE_READ reads the header information from an element file.
  • ELEMENT_WRITE writes a TRIANGLE ".ele" file.
  • I4_MAX returns the maximum of two I4's.
  • I4_MIN returns the smaller of two I4's.
  • I4MAT_COPY copies one I4MAT to another.
  • I4MAT_TRANSPOSE_PRINT_SOME prints some of an I4MAT, transposed.
  • I4VEC_COPY copies an I4VEC.
  • NODE_DATA_EXAMPLE returns the node information for the example.
  • NODE_DATA_READ reads the data from a node file.
  • EXAMPLE_NODE_SIZE returns the sizes of node information for the example.
  • NODE_SIZE_READ reads the header information from a node file.
  • NODE_WRITE writes a TRIANGLE ".node" file.
  • R8MAT_COPY copies one R8MAT to another.
  • R8MAT_TRANSPOSE_PRINT_SOME prints some of an R8MAT, transposed.
  • TIMESTAMP prints the current YMDHMS date as a time stamp.
